## Text Notes Manager
A simple text-based Notes Manager built in Java.  
It allows you to write, read, and delete notes stored in a text file using FileWriter, FileReader, and BufferedReader.
## Features
- Write Notes – Add multiple lines and save them to a file
- Read Notes – View everything saved in notes.txt
- Delete Notes – Clear the entire notes file in one click
- Uses FileWriter, FileReader, and BufferedReader
- Beginner-friendly menu interface
## About
This project demonstrates simple Java File Handling using:
- FileWriter
- FileReader
- BufferedReader
It is ideal for beginners learning file I/O operations in Java.
